来电闪光问题排解

亲爱的用户您好，感谢您使用来电闪光，若您在使用上遇到困难，请先查看以下常见问题的解决方法，祝您使用愉快。

  • 问题排解

1.通知闪光无法正常运作?

确认已开启通知存取权，点我启用

2.电池相关系统设定

请确认您的手机在省电相关的设定中排除来电闪光，并确保您的手机有足够的电力。

3.使用一阵子后就无法运作了?

部分手机系统会自动为了效能，会不定期的清除其他App的背景服务，因此来电闪光也很有可能被清理掉，因此为了确保来电闪光的服务正常运作，当您使用一阵子后无法收到讯息时，请尝试重新启动手机。

4.有什麽App会影响来电闪光运作呢?

若您有安装省电、防毒、效能优化等相关APP时，这些APP可能会为了加速系统运行，并将来电闪光的背景服务停止，请将来电闪光设定为白名单。

13.我是OPPO用户，遇到了些问题

亲爱的OPPO用户您好，因OPPO系统的特性，并不是每次都会闪光，因此也造成广大OPPO用户的困扰，但幸运的是我们提供了一些改善方法。教学以 OPPO R9s 为例，如果你使用的是其他 OPPO 机型请参考下面项目自行找寻对应设定哦，重新开机或是长时间待机后，请记得检查设定有没有跑掉喔。

步骤一：进入手机的[安全中心] → [权限隐私] → [自动启动管理] →把[来电闪光] 设为背景启动

步骤二：进入手机的[安全中心] → [权限隐私] → [悬浮视窗管理] → [来电闪光]设为允许

步骤三：进入手机的[安全中心] → 点击[一键体检] → 等待扫描完成后→ 选择[安全防护] → 点击第一项(不是按关闭键) → 进入后把[来电闪光]加入白名单

步骤四：点选 [设定] → [通知与状态列] → [通知管理] → 开启[允许通知]，允许来电闪光

步骤五：点选 [设定] → [电池] → [耗电细节] → 选择[来电闪光] → 关闭[背景冻结]、[检测到异常，自动最佳化]

步骤六：（确认软体是否正常运作）：进入手机的[安全中心] → 选择[自启动应用程式白名单] → 裡面有[来电闪光]则为已成功启动
