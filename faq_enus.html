Questions and answers

Thank you for using Flash Ring(Flash Ring), if you are facing some problems when you use Flash Ring, please check the Q & A first.

  • Start

1.Flash on notification not work?

Ensure the 「Notification Access」 feature was enabled, Click me to enable.

2.Battery-related settings

Sometimes you might find flash not work, ensure your device has enough power.

3.Flash Ring not work anymore after few days?

Some android devices will kill Apps background services automatically to enhance system performance, so Flash Ring could be killed sometimes, in order to ensure that the Flash Ring work perfectly, please try to restart your device.

4.What Apps might stop Flash Ring working?

If there are Apps about battery save,anti-virus, enhance performance or memory cleaner in your device, it might stop kill/stop Flash Ring service, so please ensure that add Flash Ring to the whitelist.

5.I am OPPO user, I am facing some problems

Dear OPPO users, according to OPPO system design, you will find that notification not always shows on notification bar, it affects many users, fortunately we provide a solution to solve it. Example for R9s, Other OPPO devices are similar way to solve it.

Step 1：Oepn [ Security Center ] → [ Privacy permission ] → [ auto-run management ] →enable [ Flash Ring ]

Step 2：Oepn [ Security Center ] → [ Floating windows management ] → enable [ Flash Ring ]

Step 3：Oepn [ Security Center ] → [ Quick scan ] → after scanned → click [ Security Protection ] → Find [ Flash Ring ] and Add to whitelist

Step 4 : Open [ Setting ] → [ Notification and status Bar ] → [ Manage notifications ] → Find [ Flash Ring ] and setting like below

Step 5 : Open system [ Setting ] → [ Battery ] → find [ Flash Ring ] and uncheck [ Freeze when in background ] and [ Automatically optimize when an abnormally is detected]

Step 6：(If you want to check Flash Ring is enable) :Open [ Setting ] → [ Security Center ] → [ Startup app whitelist ] → Find [ Flash Ring ]
