來電閃光問題排解

親愛的用戶您好，感謝您使用來電閃光，若您在使用上遇到困難，請先查看以下常見問題的解決方法，祝您使用愉快。

  • 問題排解

1.通知閃光無法正常運作?

確認已開啟通知存取權，點我啟用

2.電池相關系統設定

請確認您的手機在省電相關的設定中排除來電閃光，並確保您的手機有足夠的電力。

3.使用一陣子後就無法運作了?

部分手機系統會自動為了效能，會不定期的清除其他App的背景服務，因此來電閃光也很有可能被清理掉，因此為了確保來電閃光的服務正常運作，當您使用一陣子後無法收到訊息時，請嘗試重新啟動手機。

4.有什麼App會影響來電閃光運作呢?

若您有安裝省電、防毒、效能優化等相關APP時，這些APP可能會為了加速系統運行，並將來電閃光的背景服務停止，請將來電閃光設定為白名單。

13.我是OPPO用戶，遇到了些問題

親愛的OPPO用戶您好，因OPPO系統的特性，並不是每次都會閃光，因此也造成廣大OPPO用戶的困擾，但幸運的是我們提供了一些改善方法。教學以 OPPO R9s 為例，如果你使用的是其他 OPPO 機型請參考下面項目自行找尋對應設定哦，重新開機或是長時間待機後，請記得檢查設定有沒有跑掉喔。

步驟一：進入手機的[安全中心] → [權限隱私] → [自動啟動管理] →把[來電閃光] 設為背景啟動

步驟二：進入手機的[安全中心] → [權限隱私] → [懸浮視窗管理] → [來電閃光]設為允許

步驟三：進入手機的[安全中心] → 點擊[一鍵體檢] → 等待掃描完成後→ 選擇[安全防護] → 點擊第一項(不是按關閉鍵) → 進入後把[來電閃光]加入白名單

步驟四：點選 [設定] → [通知與狀態列] → [通知管理] → 開啟[允許通知]，允許來電閃光

步驟五：點選 [設定] → [電池] → [耗電細節] → 選擇[來電閃光] → 關閉[背景凍結]、[檢測到異常，自動最佳化]

步驟六：（確認軟體是否正常運作）：進入手機的[安全中心] → 選擇[自啟動應用程式白名單] → 裡面有[來電閃光]則為已成功啟動
